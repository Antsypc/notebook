* 显示当前服务器版本 `SELECT VERSION();`
* 显示当前日期 `SELECT NOW();`
* 显示当前用户 `SELECT USER();`

#### 创建数据库
`CREATE {DATABASE | SCHEMA} [IF NOT EXITSTS] db_name [DEFAULT] CHARACTER SET [=] charset_name;`
花括号表示必选项，中括号表示可选项。

`eg:CREATE DATABASE IF NOT EXITSTS test1 CHARACTER SET utf8;`
